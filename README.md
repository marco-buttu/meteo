# Meteo Job Server

Meteo Job Server exposes atmospheric and meteorological calculations through an
asynchronous HTTP API.

The documentation is split by role:

- [User Guide](docs/user.md): how to interact with the API.
- [System Administration Guide](docs/admin.md): how to install and operate the
  application.
- [Developer Guide](docs/developer.md): how the application is structured and how
  to extend it.
